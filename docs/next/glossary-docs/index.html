Glossary

bootstrap

An instantiation argument that defaults to true. If set to false, the internal bootstrap process will be skipped.

locals.publishUrl

Property that only is available to components during the save of an @published instance of a component. This value is determined by whatever url is determined during publishing.

_version

A data property managed by Amphora when using the Data Versioning feature. This property should not be overwritten to by any model.js or external source. If it is, data upgrades will be re-run. Please see the Data Versioning page for more information.
